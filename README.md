# Brewing Distilling + AI

AI for brewers and distillers, explained in plain language.

A small landing site, in the tone of Google's [People + AI Research](https://pair.withgoogle.com/), that demystifies AI for the brewing and distilling trade: what the technology actually is, the practical ways it can help in the brewhouse and still room, and an honest take on what it will not do.

## Pages

- **`index.html`** is the explainer landing page: what AI is in brewing terms, and how it helps (predict OG/FG/ABV, catch off-flavors early, hold batches consistent, plan production, search your own logs, free up time for the craft).
- **`literacy.html`** is an AI-literacy framework for brewers: asking better questions, giving the right brewing context, and judging whether an answer is actually reliable.

## Notes

Single-file static HTML, no build step. Type: Poppins (display) + Open Sans (body). Illustrations are inline SVG. Written for a brewing audience; the throughline is that AI supports brewing knowledge, it does not replace experienced brewers.
